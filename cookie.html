Cookie Policy

Ultima modifica: Marzo 2026  |  Versione 1.0

01 — Cosa sono i Cookie

I cookie sono piccoli file di testo che un sito web salva nel browser del visitatore. Vengono utilizzati per far funzionare il sito, ricordare le preferenze dell'utente o raccogliere informazioni statistiche.

02 — Cookie che utilizziamo

Questo sito utilizza esclusivamente cookie tecnici strettamente necessari. Non utilizziamo cookie di profilazione, tracciamento, pubblicità o analytics.

Nome Tipo Durata Finalità Fornitore
ck_consent Tecnico Persistente (localStorage) Memorizza la preferenza dell'utente riguardo al banner cookie (accettato / solo tecnici), per non mostrarlo ad ogni visita. Calipso (locale)

03 — GitHub Pages — Dati di Log

Questo sito è ospitato su GitHub Pages. GitHub Inc. non utilizza cookie propri per i siti ospitati, ma raccoglie automaticamente log di accesso al server (indirizzo IP, browser, pagine visitate, data/ora) per finalità di sicurezza e diagnostica. Questi dati non sono accessibili a Calipso Web Agency.

Per maggiori informazioni su come GitHub tratta questi dati consulta la Privacy Policy di GitHub.

04 — Google Fonts

Questo sito carica font tipografici tramite Google Fonts. Al caricamento della pagina il browser si connette ai server Google e trasmette l'indirizzo IP per ottenere i file dei font. Google potrebbe impostare cookie tecnici propri per ottimizzare il servizio. Consulta la Privacy Policy di Google per i dettagli.

Puoi gestire, bloccare o eliminare i cookie direttamente dalle impostazioni del tuo browser:

  • Google Chrome
  • Mozilla Firefox
  • Apple Safari
  • Microsoft Edge
