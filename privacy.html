01 — Titolare del Trattamento

02 — Dati Raccolti e Finalità

Questo sito web raccoglie unicamente i dati che l'utente fornisce volontariamente contattandoci via email. Non utilizziamo form di contatto integrati, sistemi di newsletter né strumenti di analytics di terze parti.

Dati raccolti via email (solo se ci scrivi):

  • Nome e cognome (se forniti)
  • Indirizzo email
  • Contenuto del messaggio

Questi dati vengono trattati esclusivamente per rispondere alla tua richiesta (base giuridica: esecuzione di misure precontrattuali su richiesta dell'interessato, art. 6(1)(b) GDPR) e non vengono condivisi con terze parti né utilizzati per finalità di marketing.

03 — GitHub Pages — Titolare Autonomo

Questo sito è ospitato su GitHub Pages, un servizio di GitHub, Inc. (88 Colin P Kelly Jr St, San Francisco, CA 94107, USA). GitHub agisce come titolare autonomo del trattamento per i dati di accesso al server.

Quando visiti questo sito, GitHub raccoglie automaticamente i seguenti dati tecnici nei propri log di server per finalità di sicurezza, diagnostica e gestione del servizio:

  • Indirizzo IP del dispositivo visitante
  • Data e ora della richiesta
  • URL della pagina richiesta
  • Browser e sistema operativo (User Agent)
  • URL di provenienza (Referrer), se presente
  • Codice di risposta HTTP

Questi dati sono trattati da GitHub Inc. in conformità alla propria Privacy Policy. Poiché GitHub ha sede negli USA, i dati vengono trasferiti verso gli Stati Uniti; GitHub aderisce alle garanzie previste dal Data Privacy Framework UE-USA.

Calipso Web Agency non ha accesso a tali log di server e non effettua alcun trattamento su di essi.

04 — Google Fonts

Questo sito carica caratteri tipografici tramite Google Fonts (Google LLC, 1600 Amphitheatre Parkway, Mountain View, CA 94043, USA). Al momento del caricamento della pagina, il browser stabilisce una connessione con i server Google, che può comportare la trasmissione dell'indirizzo IP dell'utente. Google tratta questi dati secondo la propria Privacy Policy. La base giuridica è il legittimo interesse (art. 6(1)(f) GDPR) alla corretta visualizzazione del sito.

05 — Cookie

Questo sito utilizza esclusivamente cookie tecnici necessari al funzionamento (memorizzazione della preferenza sui cookie). Non utilizziamo cookie di profilazione, tracciamento, analytics o pubblicità. Per i dettagli completa leggi la nostra Cookie Policy.
